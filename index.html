Make SVG paths more smooth

This is a simple tool, made by Adriaan, to make SVG paths more smooth. Select an SVG file, change the tolerance value, and see the difference.

It only works with paths, not with shapes. Your files stay in the browser.

Lower = keeps more detail, higher = more simplified Slide to make corners rounder
